Lets read about Python/ Flask

Flask is a Python-based web application framework. It belongs to the micro-framework category, which means it doesn't rely on external libraries. The Werkzeug WSGI toolkit and the Jinja2 template engine are the foundations of Flask. so lets understand different terms and technologies in brief:

  • Web Framework: it means it have collection of libraries and modules that enables a web application developer to write applications by simply just importing them.
  • Flask: as discussed above is a web application framework written in Python and its based on Werkzeug WSGI toolkit and Jinja2 template engine.
  • WSGI: Web Server Gateway Interface (WSGI) has been adopted as a standard for Python web application development. WSGI is a specification for a universal interface between the web server and the web applications.
  • Werkzeug: its a toolkit of WSGI which implements requests, response objects, and other utility functions.
  • jinja2: Jinja2 is a popular templating engine for Python. A web templating system combines a template with a certain data source to render dynamic web pages.
